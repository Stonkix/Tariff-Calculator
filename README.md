# Помощник подбора тарифов «1С-Отчетность»

Этот проект представляет собой интерактивный калькулятор, предназначенный для **расчета стоимости подключения** к сервису «1С-Отчетность» и автоматического формирования коммерческих предложений.

## Основные возможности

*   **Гибкие режимы расчета:** Поддержка как одиночных подключений, так и групп компаний (ГК) в «быстром» или «подробном» форматах.
*   **Автоматизация скидок:** Система автоматически рассчитывает скидки для групп компаний (при участии от 3-х организаций) в зависимости от их количества.
*   **Учет региональных особенностей:** Расчет производится на основе данных из JSON-конфигурации, учитывающей специфику тарифов для разных регионов РФ.
*   **Дополнительные услуги:** Возможность включения в расчет настройки рабочего места (Windows/MacOS), пакетов МЧД, расширений личного кабинета и многопользовательского режима.
*   **Генерация PDF:** Создание готового коммерческого предложения в формате PDF с детализацией всех услуг и контактными данными партнера.
*   **Персонализация:** Партнеры могут сохранять свои контактные данные и задавать кастомные цены на дополнительные услуги через локальное хранилище браузера (localStorage).

## Технологический стек

*   **Frontend:** Чистый JavaScript, HTML5, CSS3.
*   **Библиотеки:** Использование `html2pdf.js` для генерации документов.
*   **Шрифты:** Montserrat для современного интерфейса.
*   **Данные:** Динамическая загрузка тарифов из внешнего файла `Цены для Калькулятора 1СО.json`.

## Структура проекта

*   `calc-index.html` — основная разметка интерфейса и шаблоны компонентов.
*   `calc-script.js` — логика калькулятора, расчеты, управление состоянием и генерация PDF.
*   `calc-style.css` — адаптивные стили, обеспечивающие корректное отображение на ПК и мобильных устройствах.
*   `index.html` — выполняет перенаправление для корректной работы github pages.
